Level 0 is dangerous because it makes almost-correct choices feel identical. The correct survival posture is not bravery; it is boring consistency.

## First Minute

Stand still, choose one visual anchor, and listen for stable sound sources. Your first goal is preventing circular movement.

## Navigation Logic

If three landmarks repeat in the same order, stop and reverse for two room transitions. Then pick the first branch with different ceiling geometry.

## Resource Discipline

Walk by default. Save light for decision points, not straight corridors. Keep notes because Level 0 is visually simple enough to make memory unreliable.

## Exit Patterns

Valid exits usually create a sensory break: different floor sound, changed air pressure, altered light color, or a new mechanical rhythm.
